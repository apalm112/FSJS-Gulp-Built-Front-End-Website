## Full Stack JavaScript Program

## Project #8: Use Gulp to Build A Front-End Website
**********************************************************
* To run the script, clone the repo.
* ```cd``` into the project directory
* Run ```npm install```
* Then you can run any of the following commands:
	* ```gulp scripts```
	* ```gulp styles```
	* ```gulp images```
	* ```gulp clean```
	* ```gulp build```
	* ```gulp ```
